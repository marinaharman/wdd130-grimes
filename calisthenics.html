Calisthenics Workouts

Easy Bodyweight Exercises

PUSH

Rest 60 sec between sets

  • Incline Push-ups 3 x 8-12 reps
  • Pike Push-ups 3 x 5-8 reps
  • Bench/Chair Dips 2 x 8-10 reps

PULL

Rest 60 sec between sets

  • Incline Bodyweight Rows 3 x 8-12 reps
  • Doorframe Pulls 3 x 10-15 reps
  • Floor Superman Holds 2 x 15-30 secs

LEGS

Rest 60 sec between sets

  • Standard Air Squats 3 x 12-15 reps
  • Glute Bridges 3 x 12-15 reps
  • Reverse Lunges 2 x 8-10 reps per leg
