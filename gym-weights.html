  • Torso Twists with a Reach (Minute 2)
  • Bodyweight Squats (Minute 3)
  • Toy Soliders / Frankenstein Walks (Minute 4)
  • Light Jumping Jacks or Jumping Ropes (Minute 5)

Gym/Weights Workouts

Easy Gym/Weighted Exercises

PUSH

Rest 60 sec between sets

  • Dumbbell Bench Press 3 x 8-10 reps
  • Seated Dumbbell Shoulder Press 3 x 10-12 reps
  • Tricep Rope Pushdowns 2 x 12-15 reps

PULL

Rest 60 sec between sets

  • Lat Pulldown 3 x 8-12 reps
  • Seated Cable Row 3 x 10-12 reps
  • Dumbbell Bicep Curls 2 x 12-15 reps

LEGS

Rest 60 sec between sets

  • Leg Press 3 x 8-10 reps
  • Seated Leg Curl 3 x 10-12 reps
  • Goblet Squat 2 x 10-12 reps per leg

5min Cool Down

  • Child's Pose (Minute 1)
  • Kneeling Hip Flexor Stretch (Minute 2)
  • Seated Hamstring Stretch (Minute 3)
  • Chest Opener & Interlaced Fingers (Minute 4)
  • Deep Belly Breathing (Minute 5)
